This project simulates a pseudo-3D environment by projecting 2D map data into a first-person view using ray casting, the same core concept used in early FPS engines like Doom and Wolfenstein 3D.

It was built to explore:

How early FPS engines worked

Real-time rendering using math & geometry

Player movement, rotation, and collision detection
